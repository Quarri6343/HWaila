# Waila mod for Hytale

![preview](image/block.png)

Server side Waila Plugin. Clean implementation.

### Supported Mods:
[Multiple HUD](https://www.curseforge.com/hytale/mods/multiplehud) - must be installed if you want to use other HUD mods at the same time

### Features coming soon:  
Richer UI  
Entity health value display  
Chest contents display  
Machine progression display  
More config options  

### Commands:
/waila: toggle HUD visibility for that server (per-player config)
